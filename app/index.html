使用說明

現時計分公式是 CSPE common scale：5** = 7、5* = 6、5 = 5、4 = 4、3 = 3、2 = 2、1 = 1、U = 0。選修科必須同時選科目及等級，否則不會計入 Best 5 / Best 6。

課程卡只顯示資料集中已由官方來源抽出的統計欄位，例如 Lower Quartile、Median、Mean、Upper Quartile；缺資料會顯示 N/A，不會自動推算。

JUPAS 課程已接入 2025 官方 Admissions Scores PDF。JUPAS 分數採用院校及課程專屬公式，本頁會列出作參考，但不會直接與 CSPE common scale 作差距比較。
