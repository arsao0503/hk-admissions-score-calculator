Decision Logic

網站邏輯不是「推薦一個答案」，而是幫學生完成四個判斷

1. 興趣是否穩定

把興趣拆成活動類型，例如照顧人、拆解系統、處理數據、創作視覺、理解社會。興趣只作探索入口，不直接決定職業。

2. 科目是否合適

檢查該科實際學甚麼、如何評核、是否需要長期作品集 / 實驗 / 數學推理，以及學生能否承受學習方式。

3. 課程是否可行

用 predicted / actual DSE 成績、指定科目、multiplier、Lower Quartile / Median 判斷課程是較穩陣、接近平均或挑戰。

4. 出路是否接受

比較 programme area、專業資格、常見行業、UGC 薪資基準和院校 GES。薪資只作現實校準，不作單一排序標準。
